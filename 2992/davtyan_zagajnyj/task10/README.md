# Задача 10 — Определение IPv4 / IPv6 адреса

## Участники группы:
### **Давтян Е.К.**
### **Загайный С.О.**

## Описание решения
Программа определяет, является ли введённая строка корректным IP-адресом.

**Вывод:**
- `IPv4` — если адрес корректный IPv4
- `IPv6` — если адрес корректный IPv6
- `INVALID` — если строка не является корректным IP-адресом

**Идея:**
Используется стандартный модуль Python `ipaddress`. Он пытается распарсить введённую строку как IP-адрес.  
Если парсинг успешен — определяем версию протокола (4 или 6). Если возникает ошибка `ValueError` — выводим `INVALID`.

**Структуры данных:**
Дополнительные структуры не требуются, используется стандартный объект IP из модуля `ipaddress`.

**Сложность:**
- **Время:** $O(1)$ (проверка адреса фиксированной длины).
- **Память:** $O(1)$.
